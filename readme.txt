FakeSNI

Программа для прямого доступа к Рутрекеру. Аналогично популярной программе 
GoodbyeDPI не используются никакие сторонние сервисы, хотя применяется немного 
другой подход. Запросы маскируются путем отправки ClirentHello с поддельным 
значением поля SNI. Поскольку остальной обмен происходит через зашифрованное 
соединение этот параметр запроса клиента обычно используется чтобы однозначно 
определить адресата. Способ сработает только ваш если провайдер применяет 
фильтры DPI (а не IP). В настоящий момент почти все кампании испольуют различные 
варианты такого оборудования.

К сожалению, описанный метод невозможно реализовать путем изменения настроек 
в популярных браузерах даже с помощью расширений. Эта программа запускает 
локальный вебсервер который особым образом зашифровывает запросы к сайту. 
Браузер сможет общаться с сайтом через локальный адрес http://127.0.0.1:56789/


Параметры


-addr 127.0.0.1
  IP-адрес сетевого интерфейса который прослушивает программа. По соображениям 
  безопасности по умолчанию используется "127.0.0.1". Однако, если вы хотите 
  предоставить доступ другим компьютерам можно установить значение "0.0.0.0" 
  тогда программа будет отвечать на всех доступных интерфейсах. Это может быть 
  полезно для работы с мобильных устройств через стационарный компьютер.
  
-port 56789
  Номер сетевого порта который прослушивает программа для связи с браузером.
  
-host rutracker.org
  Доменное имя сайта к которому выполняется доступ. Если у вас заблокирован 
  IP адрес rutracker.org попробуйте подключиться к rutracker.net

-sni vk.com
  Строка отсылаемая в поле SNI. Подойдет имя любого популярного сайта или пустая 
  строка. Так же можно проверить блокировку указав реальное доменное имя.

-ignorecert
  Поропускать проверку принятого сертификата.
  
-nobrowser
  Не открывать страницу в браузере.
  

Установка


Установка программы не требуется. Скачайте архив соответсующий вашей ОС:

https://github.com/stopcenz/fakesni/releases/

Распакуйте и запустите файл. Достаточно полномочий обычного пользователя. 
После запуска программа сама откроет новую вкладку с сайтом. Если этого 
не произошло попробуйте ввести в адресной строке вашего браузера адрес 
http://127.0.0.1:56789/


Лицензия


Эта работа публикуется под лицензией СС0. 
Вы можете использовать программу без каких-либо ограничений.
